# 📸 Instagram Web Clone

**Fully functional, sleek & mobile-first Instagram web application**

## 📖 Description
A beautiful, responsive recreation of Instagram. Post photos, watch Reels, scroll endless stories, like, comment, save, explore suggestions and create new content — all working instantly in your browser. Built exactly as requested with pure HTML5, CSS3, JavaScript, jQuery & Bootstrap 5. No folders, no external links.

## ✨ Features
- 📱 100% responsive (desktop, tablet, mobile phones & every device)
- 🏠 Classic Instagram top + bottom navigation
- 📖 Horizontal swipeable Stories with LIVE badges
- ❤️ Double-tap to like + heart animation
- 💬 Real-time comments & likes with instant updates
- ➕ Create new post modal (photo picker simulation)
- 🔍 Working search bar
- 👤 Profile dropdown + fake logout
- ⭐ Suggested accounts + trending Reels teaser
- 📌 Save / bookmark posts
- 🔥 Modern gradient Instagram logo & colors
- 🚀 Zero external dependencies beyond CDNs — works offline

## 🛠️ Technologies
- **HTML5** – Semantic & accessible markup
- **CSS3** – Instagram-style gradients & animations
- **JavaScript** – Core interactivity & state management
- **jQuery** – Smooth DOM handling
- **Bootstrap 5** – Responsive grid & modals

## 🚀 How to Run
1. Download the four files (`index.html`, `style.css`, `script.js`, `README.md`)
2. Keep them all in the same folder (no sub-folders)
3. Open `index.html` in any browser
4. Start posting, liking, commenting and exploring!
